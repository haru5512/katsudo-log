# 🌱 Katsudo Log (活動記録アプリ)

日々の活動を簡単に記録・管理するためのWebアプリケーションです。
Google Apps Script (GAS) と連携して、スプレッドシートにデータを保存・同期します。

## ✨ 特徴
- 📱 スマホ対応のレスポンシブデザイン
- 🗣️ 音声入力対応 (Gemini AI連携)
- 📊 月報の自動集計
- ☁️ Googleスプレッドシート同期

## 🚀 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## 🛠️ 技術スタック
- React
- Vite
- Google Apps Script (Backend/Database)
