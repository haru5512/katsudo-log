主な機能

LogNoteでできること

記録から出力まで、活動報告に必要なすべてがひとつのアプリで完結します。

FEATURE 01
✏️
記録する

カテゴリ・時間・メモをタップ数回で入力。移動中でもサッと記録できます。スマホに最適化されたUIで、入力のストレスがゼロ。

FEATURE 02
📋
一覧で見る

日別・カテゴリ別にフィルタリング。過去の活動をすぐに振り返れます。検索・絞り込みで必要な記録に即アクセス。

FEATURE 03
📊
月次で集計

カテゴリごとの時間配分や活動回数を自動集計。自分の動きを可視化して、活動のバランスを確認できます。

FEATURE 04
📤
出力する

報告書形式・Discord形式などワンタップでテキスト出力。コピペで提出完了。月末の報告書作成が劇的に楽になります。

実際の画面

これが実際のLogNoteです

シンプルで使いやすいUIを、スマホからそのまま操作できます。

📋 一覧 — 日別・カテゴリ別で記録を確認
📊 月報 — 月次の活動ログを自動集計
📤 出力 — ワンタップで報告書テキストを生成
導入メリット

管理する側にも、
使う側にも。

LogNoteは管理者と隊員、双方の課題を同時に解決します。

🏢 管理者（あるやうむ）にとって
  • ✓ 16市町村の隊員の活動記録を統一フォーマットで把握できる
  • ✓ 月次報告の品質・粒度が揃い、自治体への報告もスムーズに
  • ✓ 隊員の活動傾向を定量的に把握でき、サポートの質が上がる
🙋 隊員にとって
  • ✓ 日々の記録が数秒で完了、月末にまとめる苦痛がなくなる
  • ✓ 自分の活動を振り返り、次のアクションに活かせる
  • ✓ スマホのホーム画面から即アクセス、アプリのインストール不要
使い方

3ステップで始められる

アカウント登録もアプリインストールも不要。URLを開くだけで今日から使えます。

1
アクセス
開くだけ

URLを開いてホーム画面に追加するだけ。アカウント登録不要。インストール不要。

2
記録
10秒で入力

カテゴリを選んで、時間とメモを入力。1件あたり10秒。移動中でもサッと記録。

3
出力
ワンタップ完成

月末に「出力」ボタンを押すだけ。報告書テキストが自動生成。コピペで提出完了。
